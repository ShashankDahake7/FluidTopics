API Reference Guide

This document covers all available API endpoints for the Fluid Topics Content Delivery Platform.

Authentication

All API requests require authentication using JWT tokens. Include the token in the Authorization header:

Authorization: Bearer <your-jwt-token>

Register

POST /api/auth/register

Create a new user account. Required fields: name, email, password.

Login

POST /api/auth/login

Authenticate and receive a JWT token. Required fields: email, password.

Content Ingestion

Upload and process documents through the ingestion pipeline.

Upload File

POST /api/ingest/upload

Upload a single file for ingestion. Supports HTML, Markdown, DOCX, XML, and ZIP formats. Maximum file size: 100MB.

Ingestion Status

GET /api/ingest/status/:id

Check the processing status of an ingestion job. Returns status (pending, processing, completed, failed) and ingestion logs.

Search

Search across all indexed topics with full-text and faceted filtering.

Full-Text Search

GET /api/search?q=query&page=1&limit=20

Search for topics matching the query. Supports fuzzy matching, highlighting, and relevance scoring.

ParameterTypeDescription
qstringSearch query
pagenumberPage number (default: 1)
limitnumberResults per page (default: 20)
tagsstringComma-separated tag filter
productstringProduct filter
sortstringrelevance, date, or views

Topics

Retrieve and browse topic content.

List Topics

GET /api/topics?page=1&limit=20

Get a paginated list of all topics with basic metadata.

Get Topic

GET /api/topics/:id

Retrieve full topic content including HTML body, metadata, and hierarchy information.

Related Topics

GET /api/topics/:id/related

Get content-based recommendations for related topics using tag overlap and product matching.

Analytics

Track user interactions and retrieve engagement metrics.

Track Event

POST /api/analytics/track

Record a user event (search, view, click, feedback).

Dashboard Stats

GET /api/analytics/dashboard?days=30

Retrieve aggregate analytics including total searches, views, top queries, and daily trends.
